This Directory contains, the templates used for creating ID cards with the Zebra p110i printer.

You need zebra card studio on your machine to print them out. The State Fair template utilizes a spreadsheet/.csv file as a "database". 
It uses the spreadsheet to bulk print your cards (Which is required for printing the 300-400 cards that you will be producing).

Templates:
PMG.car
StateFair.car

Logos:
HTC_Logo.jpg - used for both cards
StateFair_Logo.jpg - used in state fair card.
