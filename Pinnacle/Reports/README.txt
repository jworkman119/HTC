This directory contains the reports used in the application.

The reports are generated using the following:
The application creates an xml document.
The FOP is called from the application to create a pdf, from the associated *.xsl.
The pdf is pushed to the end-user.

The following tools will be needed.

FOP: http://xmlgraphics.apache.org/fop/
XML: http://xmlnotepad.codeplex.com/
NotePad++: http://notepad-plus-plus.org/

*Here is information about xsl-fo:
http://www.w3schools.com/xslfo/default.asp

* Complicated but once you get the hang of it, works very consistently.
